# IT Blog — Новинний блог про інформаційні технології

Даний проєкт розробляється в межах курсу **"SEO та пошукова оптимізація"**.

## Мета проєкту
Створення реально працюючого, SEO-оптимізованого новинного блогу в production середовищі з використанням сучасного технічного стеку.

## Тематика контенту
Блог висвітлює актуальні новини та статті за такими категоріями:
* **JavaScript / Frontend розробка:** Новини екосистеми, фреймворки та бібліотеки.
* **Backend та DevOps:** Серверні технології, архітектура та автоматизація.
* **Штучний інтелект та ML:** Огляди нейромереж та алгоритмів.
* **Кібербезпека:** Захист даних та аналіз загроз.
* **Огляди інструментів та технологій:** Тестування нового софту та заліза.

## Команда та розподіл ролей
У проєкті задіяно 4 фахівці з чітким розподілом обов'язків:

1.  **Frontend розробник - Придаткін Максим:** Відповідає за Next.js, компоненти та реалізацію SSR/SSG стратегій рендерингу.
2.  **Backend розробник - Сікора Денис:** Відповідає за REST API, проектування бази даних PostgreSQL та систему автентифікації.
3.  **DevOps & Technical SEO - Пазюк Олександр:** Відповідає за розгортання на Railway, CI/CD, налаштування доменів та технічних сторінок.
4.  **SEO Strategist & Content Manager - Житарюк Ксенія:** Відповідає за семантичне ядро, мета-дані, структуру контенту та аналітику.

## Технічний стек
* **Frontend:** Next.js (App Router) + React + SSR/SSG
* **Backend:** Node.js + Express
* **База даних:** PostgreSQL
* **Хостинг:** Railway
